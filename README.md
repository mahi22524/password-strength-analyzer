# 🔒 Password Strength Analyzer

A Python-based Password Strength Analyzer built using **Streamlit**. This application evaluates password strength, provides security suggestions, detects password reuse using SQLite, and generates strong password recommendations.

---

## 🚀 Features

- Check password length
- Check password complexity
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Password strength score
- Progress bar
- Password improvement suggestions
- Strong password generator
- Show/Hide password
- Password reuse detection using SQLite
- SHA-256 password hashing

---

## 🛠️ Technologies Used

- Python
- Streamlit
- SQLite
- Regular Expressions (re)
- hashlib (SHA-256)
- Git & GitHub

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/mahi22524/password-strength-analyzer.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
